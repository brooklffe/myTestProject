1、查看文件状态
    git status
2、全部文件增加到待提交区域
    git add .
3、提交版本
    git commit -m "v0.1"
4、版本回滚
    git reset --hard 版本唯一表示
5、回滚后的版本重新找回(找到多次修改记录)
    git reflog
    git reset --hard 回滚的版本标识
6、查看所在分支
    git branch
7、创建分支
    git branch xxxx
8、切换分支
    git checkout xxxx
9、合并分支
    git merge xx分支名称
10、git 上传
    给远程仓库起一个名字
    git remote add origin https://github.com/brooklffe/myTestProject.git
    上传本地某个分支
    git push -u origin dev
11、git 拉代码
    git clone 远程仓库地址（内部实现了别名） 第一次拉全部代码
    切换分支
    git checkout 分支
    git pull origin dev  //从origin的dev分支拉代码更新
